# deck/: Pre-Seed Pitch Deck

Self-contained HTML slide deck (13 slides). **Private**, not deployed to the
public site.

## Present
Open `deck/index.html` in a browser (full screen, F11).
- Navigate: **← / →**, **Space**, PageUp/PageDown, Home/End.
- Slide counter bottom-right.

## Export to PDF
Browser → Print → "Save as PDF", layout **Landscape**, margins **None**,
enable "Background graphics". (Each slide is one viewport, for a perfect
1-slide-per-page export, set paper size to a 16:9-ish custom size if needed.)

## Edit
Content is inline in `index.html` (one `<section>` per slide). Numbers come
from `../model/model.py` (run it, then update slide 10) and the docs:
`../docs/13-fundraising-playbook.md` (deck skeleton), `../docs/03`, `../docs/08`.
