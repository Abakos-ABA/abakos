# 06 · Go-to-Market

## Core principle

Mining reward is **subsidy**, not revenue. Real, fast revenue = selling compute + marketplace fee + Abakos Chat margin. Secure supply (GPUs) first, then steer demand onto it.

## Supply side (GPUs first)

- **Neoclouds with idle capacity** (Vast.ai/RunPod/Together-like): pitch = "extra yield per watt on unused capacity without changing your business." 1 SDK, 1 Docker image.
- **Solo miners / consumer GPUs** (3060–4090): per arXiv ROI positive → simple 1-click installer + honest reward story.

## Demand side (by speed to revenue)

1. **Latency-insensitive batch jobs** (fastest revenue): mass embedding generation, synthetic training data, LLM data labeling, image/video batch. Customers: AI startups, data teams.
2. **End users via Abakos Chat:** largest long-term lever, mainstream.
3. **IDE users (Cursor, VS Code):** copy-paste config for Continue/Cline, pitched as "30% cheaper coding AI." Viral channel (r/LocalLLaMA, HN).
4. **Indie devs & small AI startups:** same OpenAI API for apps, 20–40% below OpenRouter/AWS.
5. **Academic labs:** subsidized GPU hours = strong pull + credibility (papers, citations).
6. **Non-AI GEMM** (simulation, bioinformatics, financial Monte Carlo): later, broadens market.

## Who buys compute? (short answer to the original question)

| Buyer | Willingness to pay | Time horizon |
|---|---|---|
| AI startups / indie devs | high, if cheaper | immediate |
| Batch / data teams | high | immediate |
| End users (chat) | small per head, huge mass | medium |
| Academic researchers | medium (tight budget) | medium |
| Neoclouds (as resellers) | indirect | medium |
| Render / science compute | medium | later |
| Crypto speculators (token) | high, but unreliable | immediate (bootstrap only) |

## Acquisition tactics

- **"Useful-compute leaderboard"** public → directly against Pearl's proven idle story: "Pearl burns 112 MW for nothing. Abakos doesn't."
- **Researcher grants** from ecosystem pool → first real workloads + academic validation.
- **Abakos Chat as Trojan horse:** mainstream users bring demand without ever touching "crypto."
- **Reference partner** (1 neocloud) reproduce → case study → more partners.

## Marketing channels

- Crypto/DePIN: X/Twitter, Discord, relevant media (HashrateIndex, Alea, AGTI).
- AI devs & IDE: Hacker News, Reddit (r/LocalLLaMA, r/cursor), GitHub, HuggingFace, Continue docs/Discord.
- Mainstream (chat): classic performance marketing, app stores, SEO.
- Academic: arXiv paper/whitepaper, university partnerships.
