# Abakos (ABA): One-Pager

**Compute that pays for itself.**
The AI-compute blockchain where every *paid* GPU job also secures the chain, so compute costs less than anywhere else.

---

**Problem.** AI is bottlenecked by expensive, centralized GPU compute. Proof-of-Work chains burn huge energy for *only* security. The leading "useful work" chain (Pearl) verifies that a matmul was *correct*, not *useful*: ~320,000 GPUs, ~112 MW, **zero** real AI.

**Solution: 3 pillars.**
1. **Enforced usefulness:** full block rewards only for compute bound to a *paid* escrow job; empty mining earns 25%.
2. **On-chain marketplace + stablecoin escrow:** buyers lock payment, miners deliver output + ZK proof, settlement is atomic.
3. **Abakos Chat:** a "ChatGPT for everyone", payable with the token; mainstream demand → real, settled compute.

**Why we win (moat).**
1. **Enforced usefulness**: full rewards only for *paid* escrow work; Pearl’s network measured **0 %** useful AI (arXiv 2606.04819).
2. **2-for-1 economics**: same GPU hour earns mining subsidy *and* paid output → undercuts io.net/OpenRouter on effective price.
3. **Demand stack**: on-chain marketplace + Abakos Chat + OpenAI-compatible IDE API; no competitor combines all three with PoW subsidy.

**vs. key players.** Pearl: we enforce usefulness, not just matmul correctness. Gonka: we add PoW mining subsidy + stablecoin escrow + consumer app. GPU marketplaces: we subsidize the same watt-hour with block rewards. Central APIs: cheaper open-weight path via mining.

**Who buys compute.** AI startups & indie devs, data teams (embeddings/labeling/synthetic data), everyday users via Abakos Chat, academic labs, neoclouds reselling idle GPUs.

**Business model.** 2% marketplace fee (1% treasury / 1% burn) + Abakos Chat margin, revenue in stablecoin. ABA (2.1B max): mining rewards, payments, staking; deflationary via burn.

**Traction (early, honest).** Live hub + litepaper/whitepaper at **abakos.ai** with segmented waitlist; private monorepo with tested node economics + green CI; testnet-first plan (`docs/17-testnet-plan.md`). Next: public PoUW testnet → marketplace batch → audit → mainnet.
